# 🎯 Тестовое задание: Авторизация и список товаров

Добро пожаловать! Я успешно завершил разработку веб-приложения согласно предоставленному техническому заданию.

## ✅ Что сделано?

Это приложение обеспечивает следующие ключевые функциональные возможности:

- <b>Авторизация:</b> Пользователь проходит проверку учетных данных с помощью JWT, позволяя войти в систему и получать доступ к персонализированной информации.
- <b>Отображение товаров:</b> Список товаров представлен в виде удобных карточек с изображениями, названием, категорией и ценой.
- <b>Адаптивный интерфейс:</b> Внешний вид приложения автоматически адаптируется под экраны различных размеров, будь то десктоп или мобильное устройство.

## 💻 Как работает?

### Основные компоненты:

- <b>Страница авторизации:</b> Здесь пользователь вводит свое имя пользователя и пароль. Проверяется минимальная длина ввода (не менее трех символов). После успешной проверки пользователь перенаправляется на главную страницу.
- <b>Главная страница:</b> Показаны карточки товаров, включая возможность добавления товара в корзину (при условии авторизации).
- <b>Header:</b> Меняет внешний вид в зависимости от статуса аутентификации пользователя. Если пользователь залогинен, отображаются его имя и фамилия вместе с возможностью выйти из системы.
- <b>Footer:</b> Содержит текущий год и дополнительную информацию о залогиненном пользователе (его электронную почту).

## ⚙️ Технология и инструменты

- <b>TypeScript:</b> Используется для четкого определения типов и повышения надежности кода.
- <b>Next.js (App Router):</b> Позволяет быстро разрабатывать высокопроизводительные приложения на основе React.
- <b>Zustand:</b> Управление глобальным состоянием на клиентской стороне.
- <b>Axios:</b> Выполняет запросы к внешнему API (DummyJSON).
- <b>SCSS модули:</b> Помогают организовать стили внутри отдельных компонент.
- <b>Eslint, Prettier, Stylelint:</b> Поддерживают единый стандарт оформления и чистоты кода.
- <b>Docker:</b> Готовая сборка приложения для простого развертывания.
